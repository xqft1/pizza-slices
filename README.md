# PizzaSlices

PizzaSlices is the official NFT collection for the Pizza Protocol.

Every slice begins life as a humble `CHEESE.EXE`, but appearances don't stay that way for long. By burning Pizza tokens, holders can permanently upgrade their slice through increasingly rare variants, all the way to the legendary `GOLDEN.EXE`.

Unlike traditional NFT collections, PizzaSlices are dynamic. The artwork is determined on-chain, meaning your NFT evolves as you interact with the protocol.

## Features

- ERC721A for gas-efficient minting
- 10,000 PizzaSlices
- 200 reserved for community airdrops
- On-chain upgrade progression
- Dynamic metadata
- Nine unique slice variants
- Upgrade costs paid in Pizza tokens
- Liquidity rewards distributed to Pizza LPs
